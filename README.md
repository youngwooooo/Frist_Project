# 온라인 서점 시스템
알라딘, yes24와 같은 온라인 서점 마켓을 벤치마킹하여 온라인 서점 기능을 구현한 콘솔 프로그래밍
<br><br><br>

## ▷ 개발 환경
- Java
- Oracle
- JDBC
- Eclipes
<br><br><br>

## ▷ 담당 기능
### 1. 도서 관리
- 도서 조회·구매·구매 취소<br>
- 도서 구매 시 재고 감소, 도서 구매 취소시 재고 증가<br>
- 검색 기능(카테고리·도서명·작가명)<br>
- 도서 리뷰 조회·등록<br><br>

### 2. 공지사항
- 공지사항 조회<br><br>

### 3. 마이페이지
- 개인 정보 조회 및 수정<br>
- 도서 주문 내역 조회<br>
- 1:1 문의 조회 및 등록<br>
